获取设备信息 ​

运行 HiSuite ，根据软件指引连接设备

设备连接成功后， HiSuite Proxy 会在右侧弹出窗口，将会显示设备的 Phone Model（手机型号） 、 Phone Region（手机地区） 等信息

在手机端的拨号界面输入 *#*#2846579#*#* 打开 ProjectMenu（工程菜单）

在 2.Veneer Informations（单板信息查询） - 1.Version Info（版本信息） 下找到：

  • Base Software Version（基础包版本号）
  • Cust Software Version（定制包版本号）
  • Preload Software Version（预装包版本号）

部分系统（如 HarmonyOS ）可能需要 关于手机 - 版本号 中括号内的信息（如 C00E205 ），以此来判断应刷入哪个系统包

添加 ROM ​

访问 Firm Finder 或 Firm Finder v2.0 网站：

  • Phone Model 处填入 手机型号
  • Vendor/Country (hw-eu) 处填入 手机地区

    例：中国大陆地区（全网通）需填入 all-cn

  • Target Version 处填入需要的 系统版本号

找到需要的ROM后，点击 View Rom - Add Rom ，此时浏览器左下角应提示 ROM successfully added.

刷入 ROM ​

运行 HiSuite ，根据软件提示连接设备，点击 Update（系统更新） ，检查弹出的窗口中显示的系统是否正确

补充 ​

其他的升降级方法1 ​

Tips：
1.以下内容是在 华为官方政策支持 的情况下
2.仅适用于 当前设备的系统 不在 华为官方的维护周期内 的情况（无法通过常规手段升降级）
3.需要利用其他的同一型号设备（且 华为手机助手 支持升降级此设备），下载升降级所需的ROM包

在 华为手机助手 的 系统更新 页执行升降级操作，等到执行至 安装 步骤时停止更新操作，然后拔掉连接设备的数据线

Tips：停止更新操作有以下两种方法
1.直接点击 取消 来停止更新操作（需要等待的时间比较长）
2.在 任务管理器 中结束 HiSuite 、 HuaweiHiSuiteService 进程，来强行停止更新操作（最省时）

在资源管理器 的 地址栏 输入 %USERPROFILE%\Documents\HiSuite\ROM 并回车，然后在该路径下找到下载的ROM包，然后将其放入 U盘 / 存储卡（要使用设备支持的存储卡/U盘）

实际路径为：C(系统盘):\Users\用户名\Documents(文档)\HiSuite\ROM

其他可能可行的方案（不保证可行性）：
不动ROM包，通过数据线连接需要升降级的设备，打开华为手机助手的系统升级界面，此时应会自动检测到存放的ROM包，应该可以执行升降级操作

将需要升降级的设备关机，然后长按 音量+ + 电源键 来进入 eRecovery模式 ，根据提示来执行升级操作

不同华为设备进入 eRecovery模式 的方法不同，请根据实际情况操作！
